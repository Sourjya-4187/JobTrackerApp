# JobTrackerApp

Next.js 14 job tracker with localStorage persistence, an Add Job modal, a Job Description field, and a mobile-friendly responsive layout.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

- The app builds as a static export during GitHub Actions.
- GitHub Pages should use the `GitHub Actions` source.
- The site is built with the `/JobTrackerApp` base path for project-page hosting.
